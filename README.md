# Meteor + Vue + Vuetify
## Installation (Dev mode): 

1. Install meteor (if is not installed yet).
2. Clone repository
  ``` git clone https://github.com/Hernanm0g/meteor_vuetify.git ```
3. Navigate to directory ``` cd meteor_vuetify ```
4. run Meteor ``` meteor ```
5. Open in browser (localhost:3000)
